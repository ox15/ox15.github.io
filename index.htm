About this site

Some links have "access keys," which are keyboard shortcuts just for this website. To use them, press Alt (or Alt-Shift) on Firefox) followed by the first letter of the link (R for the "report a problem" link below, for example).

This site was built with Github Pages

This site uses Google Analytics to see how many people have visited it. Google Analytics knows what your location was when you came here, but we don't collect or use any personal information about you. We don't store cookies, but Google uses cookies on this site for analytics.

The Pong game was made with JavaScript. The Save the Sheep and Invaders games were made with Greenfoot.
